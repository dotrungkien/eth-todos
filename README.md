# a Simple Todo Dapp

## Technical stack

- Solidity
- Web3js
- Truffle
- Reactjs

## How to run

1. Run truffle develop.

```js
truffle develop
```

it will start a local blockchain at port 9545

2. Contract Migration.

```js
truffle migrate --reset
```

3. Run client.

```js
cd client/
yarn start
```

your app will run on [http://localhost:3000/](http://localhost:3000/)
